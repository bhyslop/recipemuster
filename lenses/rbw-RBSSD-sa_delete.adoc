NOTE: This operation deletes a service account from the
{rbtgi_depot_project}.
It is a multi-role operation that can be invoked by any authenticated role with IAM admin access.

. *Validate Input Parameters*:
.. {rbbc_require}
`«INPUT_SA_EMAIL»` provided as first argument
.. {rbbc_fatal}
if missing: "Service account email required"

. *Authenticate*:
.. {rbbc_require}
caller has valid OAuth token with `https://www.googleapis.com/auth/cloud-platform` scope
.. {rbbc_store}
`«ACCESS_TOKEN»` from caller context

. *Delete Service Account*:
.. {rbbc_call}
link:https://cloud.google.com/iam/docs/reference/rest/v1/projects.serviceAccounts/delete[iam.projects.serviceAccounts.delete] with:
... method: `DELETE`
... endpoint: `https://iam.googleapis.com/v1/projects/{rbrr_depot_project_id}/serviceAccounts/«INPUT_SA_EMAIL»`
... header: `Authorization: Bearer «ACCESS_TOKEN»`
.. {rbbc_require}
HTTP 200 response OR HTTP 404 (already deleted)
.. {rbbc_warn}
if HTTP 404: "Service account not found (already deleted)"

. *Display Results*:
.. {rbbc_show}
delete operation completed
