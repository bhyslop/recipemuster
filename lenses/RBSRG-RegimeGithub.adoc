// ⟦axhrb_regime⟧
{rbrg_regime}

This regime specifies variables that define GitHub authentication and identity within the Recipe Bottle system.
Each
{rbrg_regime}
instance contains the credentials needed to authenticate with GitHub for repository operations and container registry access.
RBRG assignment files are external configuration files stored outside the repository.

// ⟦axhrv_variable⟧
{rbrg_username}

GitHub username for repository and registry authentication.
Required; xname validated (1-39 characters).

// ⟦axhrv_variable⟧
{rbrg_pat}

GitHub Personal Access Token for API and registry authentication.
Required; string (40-100 characters).
Security-sensitive: masked in render output, never logged.

// ⟦axhro_kindle⟧
{rbkro_kindle}

Validates both RBRG variables inline during kindle.
Uses buv_env_xname for username, buv_env_string for PAT.

// ⟦axhro_validate⟧
{rbkro_validate}

RBRG kindle performs validation inline — no separate validate_fields function.
The kindle function is the validator for this regime.

// ⟦axhro_render⟧
{rbkro_render}

Renders via rbcr_* shared module.
Section: GitHub Identity.
RBRG_PAT displayed as masked (redacted with character count).
Validates after full render completes.
