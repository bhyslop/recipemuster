//axhob_operation
{mkr_bottle_cleanup}

NOTE: This sequence within
{opbs_bottle_start}
removes any previous container instance to ensure a clean state before launching a new
{at_bottle_container}.

//axhos_step
. *Query Container Existence*:
.. {rbbc_call}
podman container exists for
{at_bottle_container_name}
.. {rbbc_store}
`«CONTAINER_EXISTS»` as result

//axhos_step
. *Remove Previous Container* (conditional):
.. If `«CONTAINER_EXISTS»` is true:
... {rbbc_call}
podman stop with 5-second timeout for
{at_bottle_container_name}
... {rbbc_call}
podman rm --force for
{at_bottle_container_name}
if still present after timeout
... {rbbc_call}
podman container exists to verify removal
... {rbbc_require}
container completely removed from runtime
