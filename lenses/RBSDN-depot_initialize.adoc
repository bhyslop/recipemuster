//axhob_operation
{rbtgo_depot_initialize}

The
{at_operator}
completes one-time depot initialization after
{rbtgo_depot_create}
has provisioned infrastructure.
This procedure validates the
{rbra_rubric_repo_url}
for {rbtgr_rubric_repo} management and completes the GitHub OAuth authorization for the
{rbtgi_source_connection}
created by
{rbtgo_depot_create}
in PENDING state.

This procedure is idempotent — safe to re-run.
Steps already complete (URL already valid, connection already COMPLETE) are skipped.

//axhoq_precondition
[[rbtgo_depot_initialize_depot_exists]]
{rbtgo_depot_initialize_depot_exists}
* {rbtgi_depot}
exists (completed
{rbtgo_depot_create})

//axhoq_precondition
[[rbtgo_depot_initialize_governor_exists]]
{rbtgo_depot_initialize_governor_exists}
* {rbtr_governor}
created for this depot (completed
{rbtgo_governor_reset})

//axhoq_precondition
[[rbtgo_depot_initialize_gdc_configured]]
{rbtgo_depot_initialize_gdc_configured}
* {rbrr_gdc_connection_name} and {rbrr_gdc_region} configured in RBRR

//axhoq_precondition
[[rbtgo_depot_initialize_rubric_url_configured]]
{rbtgo_depot_initialize_rubric_url_configured}
* {rbrr_governor_rbra_file} contains {rbra_rubric_repo_url}

//axhoq_precondition
[[rbtgo_depot_initialize_operator_browser]]
{rbtgo_depot_initialize_operator_browser}
* Operator has browser access for GitHub OAuth authorization

//axhos_step
[[rbtgo_depot_initialize_authenticate]]
{rbtgo_depot_initialize_authenticate}
. *Authenticate as
{rbtr_director}*:
.. Execute
{rbtoe_director_authenticate}

//axhos_step
[[rbtgo_depot_initialize_load_rubric_url]]
{rbtgo_depot_initialize_load_rubric_url}
. *Load Rubric Repo URL*:
.. {rbbc_call}
{rbtoe_rbra_load}
for
{rbrr_director_rbra_file}
.. {rbbc_store}
`«RUBRIC_REPO_URL»` from {rbra_rubric_repo_url}
.. If `«RUBRIC_REPO_URL»` is empty:
... {rbbc_show}
"RBRA_RUBRIC_REPO_URL is not set in Governor RBRA file"
... {rbbc_show}
guidance for creating a rubric repository: create a repo on a git host (GitHub as reference example), generate a fine-grained PAT with Contents read/write, construct the authenticated URL as `https://x-access-token:<PAT>@<host>/<org>/<repo>.git`, and add `RBRA_RUBRIC_REPO_URL` to the Governor RBRA file
... {rbbc_fatal}
"Set RBRA_RUBRIC_REPO_URL in Governor RBRA file and re-run"

//axhos_step
[[rbtgo_depot_initialize_validate_rubric_url]]
{rbtgo_depot_initialize_validate_rubric_url}
. *Validate Rubric Repo URL*:
.. {rbbc_call}
`git ls-remote «RUBRIC_REPO_URL» HEAD` to verify reachability and credential validity
.. If `git ls-remote` fails:
... {rbbc_fatal}
"Rubric repo URL is unreachable or credentials are invalid — check RBRA_RUBRIC_REPO_URL and ensure PAT has Contents:write permission"
.. {rbbc_show}
"Rubric repo URL validated"

//axhos_step
[[rbtgo_depot_initialize_check_devconnect]]
{rbtgo_depot_initialize_check_devconnect}
. *Check Developer Connect Connection Status*:
.. Construct connection path: `projects/{rbrr_depot_project_id}/locations/{rbrr_gdc_region}/connections/{rbrr_gdc_connection_name}`
.. {rbbc_call}
link:https://cloud.google.com/developer-connect/docs/reference/rest/v1/projects.locations.connections/get[developerconnect.connections.get] for `«CONNECTION_PATH»`
.. {rbbc_require}
HTTP 200 — connection exists
.. {rbbc_store}
`«CONNECTION_STAGE»` from `.installationState.stage`
.. If `«CONNECTION_STAGE»` is `COMPLETE`:
... {rbbc_show}
"Developer Connect connection already active — skipping OAuth step"
... Go to
{rbtgo_depot_initialize_poll_connection}

//axhos_step
[[rbtgo_depot_initialize_grant_console]]
{rbtgo_depot_initialize_grant_console}
. *Grant Operator Console Access for Developer Connect*:
.. {rbbc_require}
{rbrp_operator_email} is set
.. {rbbc_call}
link:https://cloud.google.com/resource-manager/reference/rest/v3/projects/setIamPolicy[projects.setIamPolicy] for `«DEPOT_PROJECT_ID»`
.. Add `{rbrp_operator_email}` with `roles/developerconnect.admin`
.. {rbbc_show}
"Granted operator Developer Connect admin on depot project"

//axhos_step
[[rbtgo_depot_initialize_guide_oauth]]
{rbtgo_depot_initialize_guide_oauth}
. *Guide GitHub OAuth Authorization*:
.. {rbbc_store}
`«ACTION_URI»` from `.installationState.actionUri` of the connection response
.. {rbbc_require}
`«ACTION_URI»` is present and non-empty
.. {rbbc_show}
"Developer Connect connection requires GitHub OAuth authorization"
.. {rbbc_show}
`«ACTION_URI»` for {at_operator} to open in browser
.. Guide {at_operator} through the OAuth flow:
... GCP Console shows "Request GitHub OAuth token" — click "I understand and continue"
... GitHub shows "Install Google Cloud Developer Connect" — select the account/org hosting the {rbtgr_rubric_repo}
... Repository access: select "Only select repositories" and pick the {rbtgr_rubric_repo}
... Click "Install"
... GitHub redirects back to GCP; OAuth token stored in Secret Manager
.. Wait for {at_operator} confirmation

//axhos_step
[[rbtgo_depot_initialize_poll_connection]]
{rbtgo_depot_initialize_poll_connection}
. *Poll Connection Until Active*:
.. {rbbc_poll}
link:https://cloud.google.com/developer-connect/docs/reference/rest/v1/projects.locations.connections/get[developerconnect.connections.get] for `«CONNECTION_PATH»`
.. Extract `.installationState.stage` from each response
.. Poll at `RBGC_EVENTUAL_CONSISTENCY_SEC` intervals until stage reaches `COMPLETE` or `RBGC_MAX_CONSISTENCY_SEC` timeout
.. {rbbc_require}
`installationState.stage` is `COMPLETE`
.. {rbbc_fatal}
if timeout: "Developer Connect OAuth did not complete — re-open the authorization URI and try again"

//axhos_step
[[rbtgo_depot_initialize_final_validation]]
{rbtgo_depot_initialize_final_validation}
. *Final Validation*:
.. {rbbc_show}
"Running final validation..."
.. Verify {rbtgr_rubric_repo} URL: repeat `git ls-remote` call — {rbbc_require} successful exit
.. Verify Developer Connect: repeat connections.get call — {rbbc_require} `installationState.stage` is `COMPLETE`
.. {rbbc_show}
"Depot initialization complete"

//axhoo_output
[[rbtgo_depot_initialize_validated_url]]
{rbtgo_depot_initialize_validated_url}
* Validated {rbra_rubric_repo_url} — confirmed reachable via `git ls-remote`

//axhoo_output
[[rbtgo_depot_initialize_active_connection]]
{rbtgo_depot_initialize_active_connection}
* Active {rbtgi_source_connection} — `installationState.stage` is `COMPLETE`

//axhoo_output
[[rbtgo_depot_initialize_oauth_token]]
{rbtgo_depot_initialize_oauth_token}
* OAuth token automatically stored in Secret Manager (managed by Developer Connect, not by operator)

//axhoc_completion
Successful execution confirms the depot is fully initialized for
{rbtgo_rubric_inscribe}
operations.
The
{rbra_rubric_repo_url}
enables {rbtgr_rubric_repo} management and the
{rbtgi_source_connection}
enables Cloud Build to access the {rbtgr_rubric_repo}.
Proceed to
{rbtgo_rubric_inscribe}
to set up per-vessel build infrastructure.
