// ⟦axhrb_regime⟧
{rbra_regime}

This regime specifies variables that define service account authentication credentials within the Recipe Bottle system.
Each
{rbra_regime}
instance contains the minimum identity and signing material needed for a specific service account role to authenticate to GCP APIs.
RBRA assignment files are external credential files stored outside the repository, referenced by RBRR variables (RBRR_GOVERNOR_RBRA_FILE, RBRR_RETRIEVER_RBRA_FILE, RBRR_DIRECTOR_RBRA_FILE).

// ⟦axhrv_variable⟧
{rbra_client_email}

The service account email address assigned by GCP, used as the issuer (iss) when constructing JWTs for OAuth2 token exchange.
Required; must match `*.iam.gserviceaccount.com` pattern.

// ⟦axhrv_variable⟧
{rbra_private_key}

PEM-encoded RSA private key material corresponding to the service account.
Required; presence-checked for PEM header.
Security-sensitive: never logged, masked in render output.

// ⟦axhrv_variable⟧
{rbra_project_id}

The GCP project ID that owns the service account.
Required; provides scope for all API operations.

// ⟦axhrv_variable⟧
{rbra_token_lifetime_sec}

OAuth token lifetime in seconds for JWT token exchange.
Required; integer validated in range 300-3600 (typically 1800).

// ⟦axhro_kindle⟧
{rbkro_kindle}

Sets defaults for all RBRA fields.
Detects unexpected RBRA_ variables via compgen.
Builds ZRBRA_ROLLUP with masked private key.
No external regime dependencies for kindling.

// ⟦axhro_validate⟧
{rbkro_validate}

Validates RBRA_CLIENT_EMAIL format (service account email pattern).
Validates RBRA_PRIVATE_KEY contains PEM key material (presence only, never logs value).
Validates RBRA_PROJECT_ID as non-empty string.
Validates RBRA_TOKEN_LIFETIME_SEC in range 300-3600.
Dies on unexpected RBRA_ variables.

// ⟦axhro_render⟧
{rbkro_render}

Renders via rbcr_* shared module.
Sections: Service Account Identity, Authentication Material, Token Configuration.
RBRA_PRIVATE_KEY displayed as masked (redacted with character count).
Validates after full render completes.

// ⟦axhro_list⟧
{rbkro_list}

Enumerates RBRA files referenced by RBRR (governor, retriever, director).
Reports file path and existence status for each role.
