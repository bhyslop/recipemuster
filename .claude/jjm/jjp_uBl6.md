## Charter — ensoul the hierophant

Build the hierophant: the veiled Rust application that conducts the delivery ceremony,
as the theurge conducts the tests.
It owns three commands, one per ceremony:
essai — the reversible repair lap (gates, cut, prove, rig), run as many times as it takes;
ostend — the guided irreversible reveal, where the machine performs every assert and the operator's hands perform every push;
harbinger — the stranger rig against promoted public main, summonable on its own.
The command seam is the ceremony's one load-bearing boundary: reversible versus irreversible.
The app sequences proven workers and absorbs orchestration, never authority —
with the one exception ruled below: the cut, which it owns outright in Rust.
