//axhob_operation
{bust_node_regime_validate}

NOTE: Validates a single {burn_regime} profile, identified by positional
{burn_viceroyalty}, for required-field presence and {burn_platform} enum
conformance.
Read-only; no remote contact.

//axhopt_typed_parameter
{burn_viceroyalty}

//axhoq_precondition
A {burn_regime} profile with the given {burn_viceroyalty} exists in the
local node subtree.

//axhos_step
. *Load profile*: parse the profile file for the supplied
{burn_viceroyalty}

//axhos_step
. *Required fields*: confirm {burn_host}, {burn_platform}, and
{burn_workloads} are present

//axhos_step
. *Platform sanity*: confirm {burn_platform} is exactly one of
`linux | mac | cygwin | wsl | powershell | localhost`

//axhog_guarantee
The verdict reflects the on-disk state at call time; no field was
modified.

//axhoc_completion
Exit 0 when every check passes.
Non-zero with a diagnostic message identifying the first failed check.
