//axhob_operation
{bust_privilege_regime_render}

NOTE: Prints the resolved field set for a single {burp_regime} profile
identified by a positional {burp_investiture} argument.
Read-only; no remote contact.

//axhopt_typed_parameter
{burp_investiture}

//axhoq_precondition
A {burp_regime} profile with the given {burp_investiture} exists in the
local station-user subtree.

//axhos_step
. *Locate profile*: resolve the profile file for the supplied
{burp_investiture} under the station-user subtree

//axhos_step
. *Resolve fields*: parse the profile and emit every defined
{burp_regime} field assignment

//axhog_guarantee
No remote network contact occurred and no field was modified on disk.

//axhoc_completion
Exit 0 on successful render.
Non-zero when the {burp_investiture} is not found or the profile fails
to parse.
