//axhob_operation
{bust_privilege_regime_validate}

NOTE: Validates a single {burp_regime} profile, identified by positional
{burp_investiture}, for required-field presence and field-type
conformance.
Read-only; no remote contact.

//axhopt_typed_parameter
{burp_investiture}

//axhoq_precondition
A {burp_regime} profile with the given {burp_investiture} exists in the
local station-user subtree.

//axhos_step
. *Load profile*: parse the profile file for the supplied
{burp_investiture}

//axhos_step
. *Required fields*: confirm {burp_privileged_user},
{burp_privileged_key_file}, and {burp_workload_key_file} are present

//axhos_step
. *Key file readability*: confirm both {burp_privileged_key_file} and
{burp_workload_key_file} resolve to readable files on the operator's
station and are unencrypted (so `ssh-keygen -y` will not prompt)

//axhog_guarantee
The verdict reflects the on-disk state at call time; no field was
modified.

//axhoc_completion
Exit 0 when every check passes.
Non-zero with a diagnostic message identifying the first failed check.
