//axhob_operation
{bust_node_regime_list}

NOTE: Lists every {burn_regime} profile in the local station subtree
without contacting any remote node.
Read-only enumeration scoped to the current {burs_user}.

//axhoq_precondition
The station subtree at `.buk/users/{burs_user}/` exists and is readable
by the calling user.

//axhos_step
. *Enumerate local profiles*: scan the station subtree for {burn_regime}
profile files

//axhos_step
. *Display roster*: print each discovered {burn_alias} together with its
{burn_tier} classification

//axhog_guarantee
No remote network contact occurred and no profile file was modified.

//axhoc_completion
Exit 0 on successful enumeration; an empty roster is success.
Non-zero only when the station subtree cannot be read.
