//axhob_operation
{bust_caparison_linux}

NOTE: {busn_caparison} on Linux -- single-phase admin-shell ceremony
under pre-trusted key auth. {bust_handbook_linux} owns operator-manual
posture (openssh-server install on minimal distros, Tailscale install
+ first-run `tailscale up`, ssh-copy-id); caparison owns the
idempotent admin-shell remainder.

NOTE: Operations are idempotent re-applications, not nuclear teardowns
-- `systemctl enable --now` and `systemctl mask` no-op when already in
the desired state. Closes with {bust_invigilate_linux}.

The {busn_caparison} operation on Linux:

//axhopt_typed_parameter
{burp_investiture}

//axhoq_precondition
A {burp_regime} profile exists at the given {burp_investiture} and its
{burn_regime} targets a Linux node.

//axhoq_precondition
{bust_handbook_linux} operator-manual scope is complete -- Tailscale
installed and first-run-authenticated, `ssh-copy-id` run so
{burp_privileged_user} authenticates by key, sudoers NOPASSWD where
required, openssh-server installed on minimal distros. Caparison
presupposes this scope.

//axhoq_precondition
{burp_privileged_key_file} is unencrypted.

//axhos_step
. *Open admin SSH session*:
.. {busc_call}
ssh as {burp_privileged_user} with {burp_privileged_key_file}; key-auth only
.. {busc_store}
`«ADMIN_SESSION»`
.. {busc_fatal}
on connection failure -- {bust_handbook_linux} scope

//axhos_step
. *Ensure sshd enabled and active*:
.. {busc_call}
`systemctl enable --now sshd` (or `ssh` per distro convention) over
`«ADMIN_SESSION»`; idempotent
.. {busc_fatal}
on `systemctl` failure -- {bust_handbook_linux} scope (openssh-server
install is operator-manual on minimal distros)

//axhos_step
. *Mask sleep, suspend, hibernate, and hybrid-sleep targets*:
.. {busc_call}
`systemctl mask sleep.target suspend.target hibernate.target
hybrid-sleep.target` over `«ADMIN_SESSION»`; idempotent
.. {busc_fatal}
on `systemctl mask` failure -- {bust_handbook_linux} scope

//axhos_step
. *Enable and start tailscaled*:
.. {busc_call}
`systemctl enable --now tailscaled` over `«ADMIN_SESSION»`; idempotent
.. {busc_fatal}
on `systemctl` failure -- {bust_handbook_linux} scope (install +
first-run `tailscale up`)

//axhos_step
. *Post-completion check -- run invigilate-linux*:
.. {busc_call}
{bust_invigilate_linux} for this {burp_investiture}
.. {busc_store}
`«INVIGILATE_RESULT»`
.. {busc_require}
`«INVIGILATE_RESULT»` is zero
.. {busc_fatal}
on non-zero -- invigilate diagnostic surfaced verbatim

//axhog_guarantee
On success:

* Admin SSH authenticates as {burp_privileged_user} by key alone.
* sshd is enabled and active.
* `sleep.target`, `suspend.target`, `hibernate.target`, and
  `hybrid-sleep.target` are masked.
* tailscaled is enabled and running.
* {bust_invigilate_linux} confirms every posture fact end-to-end.

//axhoc_completion
Exit 0 when the post-completion {bust_invigilate_linux} passes.
Non-zero on admin SSH failure, any host-posture op failure (sshd
enable, sleep-target mask, tailscaled enable), or post-completion
invigilate failure.
