//axhob_operation
{bust_interactive_session}

NOTE: Opens an interactive SSH session to the workload account named
by a workload {burn_regime} profile.
Foreground operation; control returns to the station shell when the
operator exits the remote shell.

//axhopt_typed_parameter
{burn_alias}

//axhoq_precondition
A workload-tier {burn_regime} profile exists at the given {burn_alias}
and is currently {bust_knock}-responsive.

//axhos_step
. *Hand control to SSH*: invoke the local SSH client interactively
against the workload account

//axhos_step
. *Wait for exit*: block until the operator exits the remote shell

//axhog_guarantee
No artifact is written to {burd_output_dir} — interactive sessions
deliberately bypass dispatch capture.
No local profile is modified.

//axhoc_completion
Exit code mirrors the SSH client's exit code when the session ends
normally.
Non-zero on connection or authentication failure before the session
opens.
