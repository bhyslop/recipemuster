//axhob_operation
{opbs_bottle_start}

NOTE: This operation removes any previous container instance and instantiates and configures the service container, thereby establishing a new
{at_sessile_service}
instance.

//axhos_step
. *Verify Local Vouch Artifacts*:
.. {rbbc_require}
sentry vouch artifact present in local image store (`docker image inspect` on `{vessel}:{consecration}-vouch`)
.. {rbbc_require}
bottle vouch artifact present in local image store (`docker image inspect` on `{vessel}:{consecration}-vouch`)
.. {rbbc_fatal}
if either vouch artifact is absent: emit clear message directing operator to run summon first; no network calls attempted

//axhos_step
. *Clean Previous Instance*:
.. {rbbc_call}
{mkr_bottle_cleanup}
to remove any existing
{at_bottle_container}
.. {rbbc_require}
cleanup completed successfully

//axhos_step
. *Launch Service Container*:
.. {rbbc_call}
{mkr_bottle_launch}
to instantiate and configure new
{at_bottle_container}
.. {rbbc_require}
container launched and network configured

//axhoc_completion
Successful execution establishes a new
{at_sessile_service}
instance with proper network configuration, ready for operation.
