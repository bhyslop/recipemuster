//axhob_operation
{rbtgo_image_rekon}

NOTE: This operation lists all tags for a
{rbtga_vessel}
package in
{rbtgi_repository}
using the Docker Registry HTTP API V2.
Raw tag enumeration — no hallmark parsing or health classification.

The
{rbtr_director}
rekons image tags:

//axhopt_typed_parameter
{rbtga_vessel}
. *Validate Input Parameters*:
.. {rbbc_require}
`«INPUT_MONIKER»` provided as first argument (vessel moniker)
.. {rbbc_fatal}
if missing: "Vessel moniker parameter required"

//axhos_step
. *Authenticate as
{rbtr_director}*:
.. Execute
{rbtoe_director_authenticate}
.. {rbbc_store}
`«DIRECTOR_TOKEN»` for subsequent API calls

//axhos_step
. *Construct Registry Endpoint*:
.. {rbbc_store}
`«REGISTRY_HOST»` as `{rbrr_gcp_region}-docker.pkg.dev`
.. {rbbc_store}
`«TAG_LIST_URL»` as `«REGISTRY_HOST»/v2/{rbrr_depot_project_id}/{rbrr_gar_repository}/«INPUT_MONIKER»/tags/list`

//axhos_step
. *Fetch Tag List*:
.. {rbbc_call}
link:https://github.com/opencontainers/distribution-spec/blob/main/spec.md#listing-tags[Docker Registry API: tag list] with:
... method: `GET`
... endpoint: `«TAG_LIST_URL»`
... header: `Authorization: Bearer «DIRECTOR_TOKEN»`
.. {rbbc_require}
HTTP 200 response with JSON body containing `tags` array
.. {rbbc_store}
`«TAGS»` as array of tag strings
.. {rbbc_fatal}
if request fails: "Tag list failed with HTTP «STATUS_CODE»"

//axhoot_typed_output
{rbst_locator}
. *Display Results*:
.. {rbbc_show}
vessel moniker and tag count
.. For each tag in `«TAGS»`:
... {rbbc_show}
tag name

//axhoc_completion
Procedure completes successfully when all tags for the vessel have been listed.
