//axhob_operation
{scr_access_setup}

//axhos_step
If {rbrn_uplink_access_mode} is disabled, this sequence adds explicit DROP rules:

* RBM-EGRESS: DROP all output on the interface attached to {at_transit_network}
* RBM-FORWARD: DROP all forward from the interface attached to {at_enclave_network}

If {rbrn_uplink_access_mode} is global or allowlist, this sequence:

//axhos_step
. Network Interface Configuration:
* Disable IPv6
* Set route_localnet on the interface attached to {at_transit_network}

//axhos_step
. NAT Configuration:
* Add POSTROUTING on the interface attached to {at_transit_network}:
** Source from enclave subnet ({rbrn_enclave_base_ip}/{rbrn_enclave_netmask})
** Exclude enclave-to-enclave traffic
** MASQUERADE to the transit-side interface IP
* This MASQUERADE rule covers the outbound-from-enclave path only.
  The entry-port path requires a separate POSTROUTING MASQUERADE rule for return-path symmetry — see {scr_port_setup} MASQUERADE Rule.

//axhos_step
. Filter Configuration:
* If {rbrn_uplink_access_mode} is global:
** Accept all output on the interface attached to {at_transit_network} in RBM-EGRESS
** Accept all forward from the interface attached to {at_enclave_network} in RBM-FORWARD
* If {rbrn_uplink_access_mode} is allowlist:
** Allow sentry DNS egress: RBM-EGRESS accepts UDP/TCP port 53 to {rbrr_dns_server}
** Allow bottle DNS to sentry: RBM-FORWARD accepts UDP/TCP port 53 to {rbrn_enclave_sentry_ip}
** Block bottle DNS bypass: RBM-FORWARD drops all other UDP/TCP port 53
** For each CIDR in {rbrn_uplink_allowed_cidrs}:
*** Accept in RBM-EGRESS matching destination
*** Accept in RBM-FORWARD matching destination
