
List all
{jjdhr_heat_s}
with summary information.

{jjds_arguments}

//axd_optional axd_defaulted
* {jjda_file}

{jjds_stdout} Tab-separated values, one line per
{jjdhr_heat}:
`{jjdt_firemark}<TAB>{jjdt_silks}<TAB>{jjdhm_status}<TAB><completed><TAB><defined>`

Where:
* `<completed>` — count of paces where {jjdpe_defined} ∧ {jjdpe_resolved} (i.e., complete)
* `<defined>` — count of paces where {jjdpe_defined} (excludes abandoned)

{jjds_exit_uniform} 0 success, non-zero error.

{jjds_behavior}

. Read
{jjdgr_gallops}
from file
. Iterate
{jjdgm_order}
array in sequence (
{jjdgm_order}
is the authoritative display order; no status-based sort)
. For each
{jjdt_firemark}
in
{jjdgm_order},
look up corresponding
{jjdhr_heat}
and output TSV line:
{jjdt_firemark},
{jjdhm_silks},
{jjdhm_status},
count of paces where {jjdpe_defined} ∧ {jjdpe_resolved} (i.e., complete),
count of paces where {jjdpe_defined} (excludes abandoned)
