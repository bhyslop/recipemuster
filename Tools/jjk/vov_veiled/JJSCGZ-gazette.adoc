
//axheb_entity
{jjsz_gazette}

A two-level map from slug to lede to content, constructed with a fixed vocabulary of allowed slugs.
Used for multiline data exchange between {jjdx_tool_s} and Claude.

== Invariants

*Freeze-on-disclosure.*
A gazette has two lifecycle phases: *building* (unfrozen, accepts mutations) and *frozen* (immutable, accepts queries).
A parse-constructed gazette is born frozen.
A build-constructed gazette freezes permanently when any method discloses notice map contents to the caller.
Internal invariant enforcement during mutation does not constitute disclosure.
Mutating a frozen gazette is fatal.

*No ordering.*
The gazette has no ordering semantic — neither across slugs nor within a slug's entries.
Callers must not depend on iteration order from any method.

*Unique keys.*
Each notice is uniquely keyed by its (slug, lede) pair.
Inserting a duplicate key is fatal with a diagnostic explaining the conflict.
Lede may be absent for slug types that identify a single item; at most one lede-absent entry per slug.

== Slug Enumeration

{jjdt_slug} defines the set of valid category identifiers for gazette notices.
Each use site constructs a gazette with a vocabulary that is a subset of these values.

== Wire Format

Gazette markdown uses `#`-headers to delimit notices:

* `#` at line start marks a notice boundary
* First token after `#` (whitespace-separated) is the slug
* Remainder of the header line is the lede (may be empty)
* Content runs from the line after the header to the next `#`-header or EOF
* Leading and trailing blank lines in content are trimmed

Round-trip guarantee: for any gazette `g` constructed by parse with vocabulary `v`, `parse(v, emit(g))` produces a gazette with identical notice map contents.

== Methods

//axhems_scoped_method
=== parse

Construct a frozen gazette from raw markdown per the wire format.
All validation errors are collected and returned as a diagnostic list rather than failing on the first.
Each diagnostic is self-contained and LLM-legible: unknown slug (with near-match suggestion if applicable), malformed header (with line number), or structural violation.

//axhempm_motif_parameter axd_required axd_repeated axt_string
Vocabulary.

//axhempm_motif_parameter axd_required axt_string
Raw markdown content.

//axhemot_typed_output
{jjsz_gazette}
Frozen gazette on success.

//axhemom_motif_output axd_repeated axt_string
Diagnostic error list on failure.

//axhems_scoped_method
=== build

Construct an unfrozen, empty gazette.

//axhempm_motif_parameter axd_required axd_repeated axt_string
Vocabulary.

//axhemot_typed_output
{jjsz_gazette}
Unfrozen gazette.

//axhems_scoped_method
=== add

Add a notice to the gazette.
Fatal if frozen, if slug is not in vocabulary, or if the (slug, lede) key already exists.
