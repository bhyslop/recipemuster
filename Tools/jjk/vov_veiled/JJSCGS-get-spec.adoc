
Get raw docket text for a
{jjdpr_pace}.

*Invocation:*
----
vvx jjx_brief <CORONET>
----

{jjds_arguments}

//axd_required
* {jjdt_coronet}
— the
{jjdpr_pace}
to query

{jjds_stdout} Raw docket text, returned inline as the tool result — not written to the
{jjsz_gazette}.
A live pace emits the docket verbatim — no prefix, no trailing newline beyond the text itself.
An
{jjdpe_abandoned}
pace's output leads with an `[abandoned]` marker line and a blank line, so a dropped pace's docket cannot be misread as live.

{jjds_exit_uniform} 0 success, non-zero error.

{jjds_behavior}

. Parse
{jjdt_coronet}
. Read
{jjdgr_gallops}
. Locate
{jjdpr_pace}
by
{jjdt_coronet}
. If the pace state is
{jjdpe_abandoned},
emit an `[abandoned]` marker line followed by a blank line
. Emit `tacks[0].text` as the inline tool result
