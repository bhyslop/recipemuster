//axvd_sheaf axd_normative
This is a normative portion of the {jjs0_imprimatur}: binding — code, tests, and decisions may rely on it.
Mark a
{jjdpr_pace}
complete and commit all uncommitted changes.
Delegates to the
{jjdr_wrap}
routine.

{jjds_arguments}

//axd_required
* {jjdt_coronet}
— target
{jjdpr_pace}

//axd_optional
* `summary` (string) — wrap summary text.
If absent, defaults to "pace {silks} complete".

//axd_optional
* `spook` (string) — wrap-time friction report.
Rides the W chalk commit as a single-line `Spook:` trailer (internal newlines collapse to spaces).
If absent or empty, the trailer is `Spook: none`, so every wrap carries the line as a grep-extractable surface for later affordance tuning.

//axd_optional
* {jjda_size_limit}

{jjds_stdout} Commit hash.

{jjds_exit_enumerated}

* 0 — success
* 1 — general error (parse failure, gallops error, git error)
* 2 — size guard exceeded (use {jjda_size_limit} to override)

{jjds_behavior}

See
{jjdr_wrap}
for the full behavioral sequence.
