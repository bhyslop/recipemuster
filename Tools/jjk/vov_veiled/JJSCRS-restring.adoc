//axvd_sheaf axd_normative
This is a normative portion of the {jjs0_imprimatur}: binding — code, tests, and decisions may rely on it.

Draft multiple
{jjdpr_pace_s}
from one
{jjdhr_heat}
to another as a single atomic operation.
All
{jjdpr_pace_s}
keep their
{jjdt_coronet_s}
— immutable for life — and are re-affiliated into the destination
{jjdhr_heat};
all
{jjdkr_tack}
history is preserved.

This operation mirrors
{jjdo_relocate}
behavior but operates on multiple
{jjdpr_pace_s}
atomically.
Either all
{jjdpr_pace_s}
transfer or none do.
Unlike
{jjdo_continue}
which has specific ceremony semantics (alter, filtering by state),
{jjdo_transfer}
is a primitive bulk transfer: mechanical movement of specified
{jjdpr_pace_s}
with no additional ceremony.

— source
{jjdhr_heat}

//axd_required

— destination
{jjdt_firemark}

//axd_required

{jjdt_coronet_s}

{jjds_exit_uniform} 0 success, non-zero error.

{jjds_behavior}

.
{jjdr_load};
on failure, exit immediately with
{jjdr_load}
error status
. Parse `coronets` parameter as JSON array of
{jjdt_coronet_s}
. Verify source
{jjdhr_heat}
exists with given
{jjdt_firemark}
. Verify destination
{jjdhr_heat}
exists with given
{jjda_to}
{jjdt_firemark}
. Verify source and destination are different
{jjdhr_heat_s}
. For each
{jjdt_coronet}
in `coronets` array, verify the
{jjdpr_pace}
exists in source
{jjdhm_paces}
. For each
{jjdt_coronet}
in `coronets` array (preserving order):
.. Move the
{jjdpr_pace}
entry from source
{jjdhm_paces}
to destination
{jjdhm_paces}
under the same
{jjdt_coronet}
— never re-keyed, never re-minted; all
{jjdkr_tack}
data is carried unchanged
.. Remove the
{jjdt_coronet}
from source
{jjdhm_order}
and append it to destination
{jjdhm_order}
.. Record the mapping — old and new
{jjdt_coronet}
identical, the unchanged identity — for output
.
{jjdr_save}
. Construct output JSON with source info, destination info, and drafted array
. Output JSON to stdout

*Validation errors:*

[cols="2,3"]
|===
| Condition | Error

| Source {jjdhr_heat} not found
| "Heat {firemark} not found"

| Destination {jjdhr_heat} not found
| "Heat {firemark} not found"

| Source and destination same
| "Cannot relocate paces to same heat"

| Invalid JSON in `coronets`
| "Invalid JSON in coronets parameter"

| {jjdt_coronet} not array
| "Expected JSON array of coronets"

| {jjdpr_pace} not found in source
| "Pace {coronet} not found in heat {firemark}"

| Empty `coronets` array
| "No paces specified for relocate"
|===
